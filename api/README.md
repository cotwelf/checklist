# projects

## 1.先有 projects 之后才可以创建 plan

## 2.表结构

| 字段名 | 类型（格式） | 说明 | 规则 |
| :------: | :----------- | :----------------------------------- |：--|
| id | 主键，int | 不允许重复 |1|
| status | int | 1：正在进行；10：已完成；9：半途而废 |手动关闭项目，状态置为 9，end_at 变为关闭当日|
| name | string | 不允许重复 |1|
| start_at | 短日期 | 创建日期，项目开始时间 ||
| end_at | 短日期 | 项目结束时间 |项目到此时间则结束，状态为 10|

## 3.表关联

projects.id = plans.p_id
